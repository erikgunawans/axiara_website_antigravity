## Brand Compliance Checks

Every page must pass `python3 execution/validate_html.py` (9 checks):

1. ✅ HTML5 structure, lang, viewport
2. ✅ Google Fonts: Outfit + Playfair Display + JetBrains Mono
3. ✅ Link to `axiara.css`
4. ✅ No emojis in visible text
5. ✅ `border-radius: 0px` everywhere (no `rounded-*` Tailwind classes)
6. ✅ Only `fade-in-up` animations (no bounce, spin, zoom, elastic)
7. ✅ Proper heading hierarchy (no skipping levels)
8. ✅ Accessibility: alt on images, labels on inputs
9. ✅ `prefers-reduced-motion` media query present

### Browser Testing

- Open each page in browser and verify:
  - Dark theme renders correctly (obsidian bg, white/silver text)
  - Frosted glass cards have blur effect
  - Ghost buttons have hover crimson glow
  - Fade-in-up animations trigger on scroll

  - All inter-page links resolve correctly
